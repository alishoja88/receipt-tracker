# Receipt Tracker Backend

NestJS backend for Receipt Tracker application.

## Setup

```bash
npm install
```

## Development

```bash
npm run start:dev
```

## Build

```bash
npm run build
npm run start:prod
```

## Testing

```bash
# Unit tests
npm test

# E2E tests
npm run test:e2e

# Coverage
npm run test:cov
```

## Environment Variables

Create `.env` file with required variables (see `.env.example`).
